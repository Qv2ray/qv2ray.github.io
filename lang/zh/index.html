Qv2ray 是一个使用 Qt 编写的跨平台的 v2ray 图形前端。

目前支持的版本是 v2.7.0-pre2。

# 🚀 起步

对于大多数用户，快速上手 将会带领他们在短时间内上手 Qv2ray。

对于每一种功能对应的详细文档，请参阅 使用手册。

# 👷 贡献

欢迎参与该项目的贡献中来！就算你不懂如何写代码，也还是有许多种方法可以帮助完善 Qv2ray 的。这里有一些例子：

  1. 🌐 翻译。帮助 Qv2ray 翻译成更多语言的版本，或者提高目前已有的翻译的质量。你可以在 Crowdin (opens new window) 上找到我们。
  2. 🐛 报告 Bug。报告 Qv2ray 的 bug 或其他使用时产生的问题，请给我们提交 issue (opens new window)
  3. 💡 新功能建议。GitHub - Qv2ray/Qv2ray (opens new window)
  4. ⚙️ 帮助开发 Qv2ray。克隆仓库，签出代码，提交高质量的 Pull request！
  5. 📆 社区维护。帮助我们创建、维护 Qv2ray 的社区！
  6. ...你还可以做更多！
